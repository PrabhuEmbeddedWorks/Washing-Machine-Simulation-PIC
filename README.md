# PIC16F877A Washing Machine Simulation

This repository contains a **simulation of an Automatic Washing Machine** implemented using the **PIC16F877A microcontroller**.  
The project is developed in **Embedded C** using **MPLAB X IDE** and simulated using **PicSimLab**.

## Project Features

- Software-controlled Power ON using keypad
- Power-on visualization using LCD loading animation
- Washing program selection
- Water level selection
- Start, Pause, and Resume functionality
- Timer-based wash, rinse, and spin operations
- Door safety mechanism
- Buzzer indication on program completion

---

##  Concepts Used

- PIC16F877A Microcontroller
- Embedded C Programming
- LCD Interfacing (HD44780)
- Keypad Interfacing using PORTB
- Timer2 and Interrupt Service Routine (ISR)
- State Machine Based Control Logic
- Real-time Embedded System Design
